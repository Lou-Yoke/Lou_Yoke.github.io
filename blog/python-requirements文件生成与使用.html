      # python requirements文件生成与使用

      在写python程序的时候，我们经常的会下载很多的模块，当我们编写完成后，准备在其他设备上部署的时候，那么新设备上需要安装我当前环境下的所有包，如果一个个找显然是挖坑埋自己，于是我们可以通过pip生成一个requirement.txt的文件，在新环境中通过读取这个文件中的模块名称进行安装环境。

      # 创建requirements.txt

      生成requirements.txt

      pip freeze > requirements.txt

      查看生成requirements.txt

      ​

      cat requirements.txt

      # 在新环境中通过此文件可以直接安装环境

      pip install -r requirements.txt

      完成安装后，新环境中就可以运行你的代码喽
